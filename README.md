## Meetups
- [Halifax dbt Meetup](https://www.meetup.com/halifax-dbt-meetup/) - Data meetup focused on dbt (data build tool), Halifax based but with virtual attendance option
- [Halihax](https://www.halihax.com/) - Halifax tech and data community and meetups
- [Data For Good Maritimes](https://maritimes.dataforgood.ca/) - Datathons for social good
